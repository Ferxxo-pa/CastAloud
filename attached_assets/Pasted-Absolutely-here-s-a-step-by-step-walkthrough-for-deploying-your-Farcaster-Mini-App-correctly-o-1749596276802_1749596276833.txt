---

## 🛠 Step-by-Step: Make Cast Aloud Work as a Farcaster Mini App

---

### ✅ 1. **Create `.well-known/farcaster.json`**

#### 📁 Step 1.1: In Replit, create a new folder called:

```
.well-known
```

> Right-click in the file tree > "New Folder" > name it **`.well-known`**

#### 📝 Step 1.2: Inside that folder, create a file called:

```
farcaster.json
```

#### 📄 Step 1.3: Paste the following JSON:

```json
{
  "frame": {
    "version": "vNext",
    "name": "Cast Aloud",
    "homeUrl": "https://castaloud.replit.app/",
    "iconUrl": "https://castaloud.replit.app/icon.png",
    "splashImageUrl": "https://castaloud.replit.app/splash.png",
    "requiredChains": [],
    "requiredCapabilities": []
  }
}
```

> ✅ Make sure the icon and splash URLs exist (or replace them with working placeholders like a logo or banner)

---

### ✅ 2. **Serve Static Files in Replit**

You need to tell Replit to serve the `.well-known` directory so Warpcast can read your JSON.

#### 🧠 If you're using Express.js:

In your `index.js` or `server.js`, add this line at the top (after your `express()` call):

```js
app.use('/.well-known', express.static('.well-known'));
```

This lets `https://castaloud.replit.app/.well-known/farcaster.json` be accessible.

---

### ✅ 3. **Restart or Re-run the App**

Click the green **Run** button in Replit to restart your server and apply the changes.

---

### ✅ 4. **Test the Manifest**

In your browser, go to:

```
https://castaloud.replit.app/.well-known/farcaster.json
```

You should see the JSON file appear — this is what Warpcast looks for.

---

### ✅ 5. **Test It in Warpcast**

Go to [warpcast.com](https://warpcast.com) and create a cast like this:

```
🎧 Try Cast Aloud – reply to any cast with your voice  
https://castaloud.replit.app
```

Click the link inside Warpcast → it should now open your Mini App in a modal inside the app. 🎉
